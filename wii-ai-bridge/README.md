## üîß Getting Started

### Prerequisites
- Nintendo Wii console (homebrew enabled)
- DevkitPro toolchain
- Python 3.8+ with AI/ML libraries
- Node.js for web tools

### Installation
```bash
# Clone the repository
git clone https://github.com/ProCityHub/AGI.git
cd AGI/wii-ai-bridge

# Install dependencies
npm install
pip install -r requirements.txt

# Setup Wii development environment
./setup-devkit.sh

# Build AI models
python train_models.py

# Compile Wii applications
make all
```
